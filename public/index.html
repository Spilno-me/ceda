What is Herald MCP?

Herald MCP is the Model Context Protocol interface that connects AI assistants like Claude Desktop to CEDA's cognitive pipeline. It enables natural language module generation, pattern learning, and cross-project context sharing.

● Generate structured modules from natural language
● Learn patterns from your work automatically
● Share context across projects and teams
● Git-verified identity for secure pattern isolation
